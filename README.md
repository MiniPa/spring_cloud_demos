#申明 
spring cloud 是一系列框架的有序集合，利用Spring Boot的开发便利性巧妙地简化了分布式系统基础设施的开发。
  如服务发现注册、配置中心、消息总线、负载均衡、断路器、数据监控等，都可以用Spring Boot的开发风格做到一键启动和部署。
  是一套简单易懂、易部署和易维护的分布式系统开发工具包。
  
  本案例展示spring集成的各种开发技术用spring boot来实现的方式。
